# 顾齐光六关科目PK

一个点开即玩的网页格斗小游戏，入口文件是 `index.html`。

## 在线发布

推荐使用 GitHub Pages：

1. 在 GitHub 新建一个公开仓库。
2. 上传本目录全部文件，包括 `.github/workflows/pages.yml`。
3. 进入仓库的 `Settings` -> `Pages`。
4. 在 `Build and deployment` 中选择 `GitHub Actions`。
5. 回到 `Actions`，等待 `Deploy static game to GitHub Pages` 执行完成。
6. 发布地址会显示在 `Settings` -> `Pages` 页面。

## 本地游玩

直接双击 `index.html` 即可开始游戏。

## 操作

- `←` / `→`：左右移动
- `↑`：跳跃
- `↓`：下蹲
- `D`：出拳
- `F`：出脚
- `空格` 连按两次后按 `D`：空中砸击
- `S`：能量满后发射 RPG
